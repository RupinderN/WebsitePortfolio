Projects

Movie Magnet NodeJS
August-September 2019

This is a web application using the MovieDB API and the 'imdb' node package to find movies that currently playing in theatres, and display their IMDB ratings. If a user creates an account, they will obtain the additional functionality of being able to filter movie ratings based on their personal taste, and can also receive emails when a movie above their specific criteria is released in theatres. It runs on NodeJS backend, and the Express.js framework, with various asynchronous calls resolved with Promises and async/await functions.

Discover More NodeJS
October 2019

This is a web application that uses the popular Spotify API to find related artists a user would enjoy based on their favourite artists. A user then has the ability to follow these artists. The special functionality of the app comes based on the fact that it only displays artists that have an arbitrary popularity rating under 65 on Spotify (this is a property for artists on the API), so the artists displayed will be relatively unpopular. It also runs on NodeJS backend, and the Express.js framework, with various asynchronous calls resolved with Promises and async/await functions.
